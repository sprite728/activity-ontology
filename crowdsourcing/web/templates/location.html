You will be given a location type (e.g., Hospital, Aquarium, etc.) and a description of that location. Write down an activity or event that can occur in that location; that is, write a verb (e.g., go, swim, eat, etc.), and an object (e.g., home, food, etc.) in the two forms.

Some examples to give you a better idea: at a bowling alley you can go bowling; at a hospital you can visit doctor.
